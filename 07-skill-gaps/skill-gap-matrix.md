# Skill Gap Matrix

## Assessment Methodology

Skill levels are estimated based on CMMI Level 3 baseline, product vintage, and technology signals from public sources. **Validate all estimates using the skills matrix survey (distribute Week 1).**

### Proficiency scale
| Level | Description |
|-------|-------------|
| 1 — Absent | Skill not present in team |
| 2 — Beginner | Awareness; some individuals exploring |
| 3 — Practitioner | Applied on projects but inconsistently |
| 4 — Proficient | Consistent application; team-wide competency |
| 5 — Expert | Deep expertise; can coach others; drives innovation |
